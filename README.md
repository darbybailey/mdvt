# Multilingual Dubbing Visualization Tool (MDVT)

**Enhance dubbing workflows with real-time NLP and interactive dashboards**


## Overview
MDVT helps content producers and localizers manage multilingual dubbing projects with precision and speed. It features NLP-powered translation, real-time sentiment analysis, and an interactive D3.js dashboard.



## Features
- Semantic NLP input for dubbing logic
- D3.js real-time visualization dashboard
- Interactive Flywheel control for adjusting processing speed
- Sentiment-aware tone matching for translated voiceovers



## Technologies Used
- Python 3.10+
- FastAPI
- D3.js
- TypeScript
- NLTK / SpaCy



## Getting Started
1. Clone this repository
2. Run `pip install -r requirements.txt`
3. Start the API with `uvicorn src.api.endpoints:app --reload`
4. Launch the frontend with `npm start` from `src/frontend`
